Python可变对象和不可变对象

2020-04-22
Word count: 631   |   Reading time≈ 2 min

概念

  • 首先要理解什么是可变对象和不可变对象，可变对象是指这个对象（变量）本身可以修改，而不可变对象一旦创建之后不可修改，只可重新赋值。Python中的可变对象和不可变对象有：
  • 可变对象：dict，list，set
  • 不可变对象：tuple，string，int，float，bool
    • 例如，创建一个可变对象dict = {1,2,3}，我可以修改他某一个值或者append一个值，dict[0] = 1 是允许的；但是当一个不可变对象tuple = （1,2,3），是不可以修改他里面的值或者增加值的，只能另外赋值tuple = （1,2,4），这个操作意思是tuple不再指向（1,2,3）这个对象了，而指向（1,2,4）这个对象的地址，简单易懂？

地址

  • 可变对象的值改变后地址不变。若两个对象指向同一个地址时，ａ值改变ｂ值也会改变（地址不变）。
  • 不可变对象的值改变后指向的地址也改变了（指向新的值所在的地址，原值还在）。若两个对象指向同一个地址，ａ值改变（另外赋值），ｂ的值和地址都跟原来一样不变。

作为函数参数传入

  • Python中的参数传递中是引用传递，即传递实参的地址给形参。
  • 当传递的参数是可变参数时，改变函数内部的形参也会改变外面被传递的实参，我的理解为在传递时，形参指向实参所指向的地址，可变参数是可修改的，且形参和实参指向的地址是同一个，一个修改时另一个也同样变了。
  • 当传递的参数是不可变参数时，形参同样指向实参所指向的地址，只不过在进行修改操作时，由于不可变参数直接不可修改，形参会指向另一个地址，地址的值为修改的值。所以现在形参和实参所指向的地址不是同一个，自然形参的地址和值不会有影响。

函数默认参数

  • 一定要设定为不可变参数
  • 函数定义时默认参数就确定下来了，之后每次调用函数时默认参数都是同一个（理解为同一个地址），如果是可变参数时可能上一次的调用会修改地址中的值（比如append）。
